fireworks.ic

Initial conditions utilities , (fireworks.ic)

This module contains functions and utilities to generate initial conditions for the Nbody simulations. The basic idea is that each function/class should returns an instance of the class Particles

Functions

ic_random_normal

Generate random initial condition drawing from a normal distribution (centred in 0 and with dispersion 1) for the position and velocity.

ic_random_uniform

Generate random initial condition drawing from a uniform distribution (between upper and lower boundary) for the mass, position and velocity.

ic_two_body

Create initial conditions for a two-body system.
