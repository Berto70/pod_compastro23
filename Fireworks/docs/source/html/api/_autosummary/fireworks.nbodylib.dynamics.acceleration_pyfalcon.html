fireworks.nbodylib.acceleration_pyfalcon

fireworks.nbodylib.dynamics.acceleration_pyfalcon(particles: Particles, softening: float = 0.0) → Tuple[ndarray[Any, dtype[float64]], ndarray[Any, dtype[float64]] | None, ndarray[Any, dtype[float64]] | None][source]

Estimate the acceleration following the fast-multipole gravity Dehnen2002 solver (https://arxiv.org/pdf/astro-ph/0202512.pdf) as implementd in pyfalcon (https://github.com/GalacticDynamics-Oxford/pyfalcon)

Parameters:

  • particles – An instance of the class Particles

  • softening – Softening parameter

Returns:

A tuple with 3 elements:

  • Acceleration: a NX3 numpy array containing the acceleration for each particle

  • Jerk: None, the jerk is not estimated

  • Pot: a Nx1 numpy array containing the gravitational potential at each particle position
