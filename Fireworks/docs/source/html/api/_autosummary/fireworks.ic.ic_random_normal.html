fireworks.ic_random_normal

fireworks.ic.ic_random_normal(N: int, mass: float = 1) → Particles[source]

Generate random initial condition drawing from a normal distribution (centred in 0 and with dispersion 1) for the position and velocity. The mass is instead the same for all the particles.

Parameters:

  • N – number of particles to generate

  • mass – mass of the particles

Returns:

An instance of the class Particles containing the generated particles
