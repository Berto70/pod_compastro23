fireworks.nbodylib.potentials

Collection of classes to estimate the Gravitational forces from potentials (fireworks.nbodylib.potentials)

This module contains a collection of classes and functions to estimate acceleration due to gravitational forces of a fixed potential

In Fireworks that functions to estimate the acceleration depends only by the particles and the softening. However, the potential can depend on a number of additional parameters. Therefore instead of creating directly the function each Potential is created through a calls derived from the basic class Potential_Base, this class returns the acceleration through the method acceleration(). This is method is just a wrapper of the specific method _acceleration() that needs to be implemented for each potential. So a Potential needs to add an __init__ method that takes into account all the parameters necessary to define the potential and the method _acceleration to properly estimate and return the acceleration. Then this method can be used inside the fireworks integrators (fireworks.nbodylib.integrators)

Example

>>> from fireworks.nbodylib.potentials import Example_Potential
>>> pot = Example_Potential(par1=1,par2=2) # Instantiate the potential
>>> accfunc = pot.acceleration # This method can be used with the fireworks integrators
